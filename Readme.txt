Singly Linked List & Serialization

This program has method definitions for different methods of a singly-linked list such as append, prepend, insert,
delete, contains, indexof, etc. It has test cases for these methods.

The second part of this program is Serialization and testing serialization with the sigly linked list. Password
is not serialized.

The program can be run by running a JUnit test on the LinkedList and Serilazation.
